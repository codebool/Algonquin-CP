#1. Files Included
1) index.html: Contains the HTML and JavaScript code to load the map.
2) styles.css: Contains custom styles for your map. Make sure this file exists even if it's empty to avoid loading errors.

#2. Open the Map in a Browser:
1) Open your web browser.
2) Navigate to the folder where your index.html file is saved.
3) Double-click the index.html file to open it in your browser. Alternatively, you can drag and drop the file into your browser's window.

#3. Interacting with the Map:
1) Once the map is loaded, you will see markers at the specified locations.
2) Click on any marker to see a popup with the address and additional information.
3) You can also click anywhere on the map to display a popup showing the coordinates of the clicked location.
