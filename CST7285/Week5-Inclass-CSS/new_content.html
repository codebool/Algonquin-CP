What is CSS?

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in HTML or XML.

CSS Syntax

A CSS rule-set consists of a selector and a declaration block. The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

<selector> {
    property: value;
    property: value;
}

Style Location

Styles can be applied in three different ways:

  • Inline Style: Using the style attribute directly in an HTML element.
  • Internal Style Sheet: Using the <style> element within the <head> section of an HTML document.
  • External Style Sheet: Using a separate CSS file linked with the <link> element.
<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        p { color: blue; }
    </style>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <p style="color: red;">This is an inline style.</p>
    <p>This is styled with internal style sheet.</p>
    <p>This is styled with external style sheet.</p>
</body>
</html>

Selectors

Selectors are used to select the HTML element(s) you want to style.

  • Type Selector: Selects all elements of a given type. Example: p { color: red; }
  • Class Selector: Selects all elements with a given class. Example: .class { color: red; }
  • ID Selector: Selects a single element with a given id. Example: #id { color: red; }
  • Universal Selector: Selects all elements. Example: * { color: red; }
  • Attribute Selector: Selects elements based on an attribute or attribute value. Example: input[type="text"] { color: red; }
  • Grouping Selector: Selects multiple elements. Example: h1, h2, p { color: red; }
<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        p { color: red; }
        .example { color: green; }
        #unique { color: blue; }
        * { font-family: Arial; }
        input[type="text"] { border: 1px solid black; }
        h1, h2, p { margin: 10px 0; }
    </style>
</head>
<body>
    <p class="example" id="unique">This is a paragraph with multiple selectors applied.</p>
    <input type="text" value="Text input">
</body>
</html>

This is a paragraph with multiple selectors applied.

Classes, IDs, and Other Styling Methods

Classes and IDs are used to apply styles to specific elements. A class can be used on multiple elements, while an ID should be unique within a page.

  • Class: Apply styles to multiple elements with the same class name. Example: .class { color: red; }
  • ID: Apply styles to a single element with a unique ID. Example: #id { color: red; }
<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        .class-example { color: green; }
        #id-example { color: blue; }
    </style>
</head>
<body>
    <p class="class-example">This is styled with a class.</p>
    <p id="id-example">This is styled with an ID.</p>
</body>
</html>

Flex Layout

CSS Flexbox Layout is a simpler layout system, typically used for parts of a web page, not the whole page. It allows you to create flexible and responsive layouts.

<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        .container {
            display: flex;
            flex-direction: row;
            flex-wrap: wrap;
            justify-content: space-around;
        }
        .item {
            background-color: lightblue;
            margin: 10px;
            padding: 20px;
            flex: 1;
        }
    </style>
</head>
<body>
    <div class="container">
        <div class="item">Item 1</div>
        <div class="item">Item 2</div>
        <div class="item">Item 3</div>
    </div>
</body>
</html>

Responsive Web Design

Responsive Web Design (RWD) refers to designing websites that adapt gracefully to different screen sizes. It involves using flexible grids, flexible images, and CSS media queries.

<!DOCTYPE html>
<html lang="en">
<head>
    <style>
        .container {
            width: 100%;
            margin: 0 auto;
        }
        @media (min-width: 600px) {
            .container {
                width: 80%;
            }
        }
        @media (min-width: 900px) {
            .container {
                width: 60%;
            }
        }
    </style>
</head>
<body>
    <div class="container">
        <p>This container is responsive to screen size.</p>
    </div>
</body>
</html>

This container is responsive to screen size.

Bootstrap

Bootstrap is a free front-end framework for faster and easier web development. It includes HTML, CSS, and JavaScript components for creating responsive, mobile-first projects on the web.

<!DOCTYPE html>
<html lang="en">
<head>
    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
</head>
<body>
    <div class="container">
        <h1 class="text-center">Hello, Bootstrap!</h1>
        <button class="btn btn-primary">Primary Button</button>
    </div>
</body>
</html>
