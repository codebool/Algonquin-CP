Introduction to Statements

JavaScript statements are composed of expressions, control structures, and function declarations. Common control structures include if-else statements and loops.

if (condition) {
    // code to execute if condition is true
} else {
    // code to execute if condition is false
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

let x = 10;
if (x > 5) {
    console.log("x is greater than 5");
} else {
    console.log("x is less than or equal to 5");
}

for (let i = 1; i <= 5; i++) {
    console.log(i);
}
