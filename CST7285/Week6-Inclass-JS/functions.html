Introduction to Functions

Functions in JavaScript allow you to encapsulate code for reuse. Functions can take parameters and return values.

function greet(name) {
    return "Hello, " + name;
}

console.log(greet("John"));

function add(a, b) {
    return a + b;
}

let sum = add(5, 3);
console.log("Sum: " + sum);
